# Gerador de QR Code

## 📌 Descrição
Projeto simples de um gerador de QR Code desenvolvido utilizando **HTML**, **CSS** e **JavaScript** em um único arquivo.  
A aplicação permite gerar QR Codes a partir de textos ou links digitados pelo usuário, com exibição imediata no navegador.

O projeto utiliza uma biblioteca externa para realizar a criação dos QR Codes de forma prática e eficiente.

---

## 🚀 Funcionalidades

- Gerar QR Codes instantaneamente
- Suporte para textos e URLs
- Interface simples e responsiva
- Atualização dinâmica no navegador
- Implementação em arquivo único

---

## 🛠️ Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Biblioteca:
  - `qrcodejs`
